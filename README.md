# Grocery App

A Flutter application for grocery shopping with payment and order tracking features.

## Prerequisites

- Flutter SDK (stable channel)
- Dart SDK is bundled with Flutter

## Quick Start

Run these commands to build and start the app:

```powershell
git clone <your-repo-url>
cd Grocery-app--main/Grocery-app--main
flutter clean
flutter pub get
flutter run -d windows
```

If you want to run on web or mobile:

```powershell
flutter run -d chrome   # Web
flutter run -d edge     # Web (Edge)
flutter run -d android  # Android (device/emulator required)
flutter run -d ios      # macOS with Xcode required
```

## Supported Platforms

- Windows (Desktop)
- Chrome (Web)
- Android and iOS (via Flutter)
